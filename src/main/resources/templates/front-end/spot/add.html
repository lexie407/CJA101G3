edit_location 景點資訊

請填寫完整的景點資訊

info 基本資訊

place
請使用正式或廣為人知的名稱，最多100個字元
location_on
請輸入詳細地址或地區，最多100個字元
public
選擇景點所在的縣市地區
phone
請輸入景點的聯絡電話，格式如：02-12345678
language
請輸入景點的官方網站URL，必須包含http://或https://
0/500
請輸入5~500字，讓其他旅人更了解這個景點
cloud_upload

點擊或拖拽圖片到此處

支援 JPG、PNG 格式，單張不超過2MB

您可以上傳多張圖片來展示景點的不同角度和特色
arrow_back 返回列表

lightbulb 景點提交須知

edit 命名建議

  • 使用正式或廣為人知的名稱
  • 避免使用個人化的暱稱
  • 如有多個名稱可在描述中補充

description 描述撰寫

  • 描述景點的特色和亮點
  • 提及適合的遊覽時間和季節
  • 說明交通方式或停車資訊
  • 分享個人體驗和推薦理由

verified 審核流程

  • 提交後會進入待審核狀態
  • 管理員會在3-5個工作天內審核
  • 審核通過後會自動上架顯示
  • 如被退回會收到通知說明原因
